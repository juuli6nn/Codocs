# <h1 align="center">Codocs</center>

###  <h3 align="center">Google Docs Code Formatting and Syntax Highlighting Extension</center>

---
## Description
Codocs is a Google Chrome extension that brings professional code formatting directly into Google Docs. 
It transforms plain code into beautifully formatted, syntax-highlighted code blocks with IDE-inspired dark themes, making 
technical documentation, assignments, reports, and code sharing significantly more readable. 

---

## Features

### Code Formatting

* **Syntax Highlighting:** Automatically detects and highlights code syntax for improved readability.
* **Dark IDE Themes:** Converts code into visually appealing dark-themed blocks inspired by modern code editors.
* **One-Click Formatting:** Instantly transform pasted code into structured, presentation-ready code blocks.
* **Google Docs Integration:** Works directly inside Google Docs without requiring external tools or websites.

### Developer-Friendly Design

* **Language Detection:** Supports syntax highlighting across multiple programming languages.
* **Consistent Styling:** Maintains clean formatting for documentation, reports, tutorials, and academic submissions.
* **Optimized Rendering:** Produces professional-looking code blocks suitable for sharing and collaboration.

---

## Installation

### From Chrome Web Store

1. Navigate to the Chrome Web Store listing.
2. Click **Add to Chrome**.
3. Confirm the installation prompt.
4. Open any Google Docs document and start formatting code.

---

## How to Use

1. Open a Google Docs document.
2. Paste or select the code you want to format.
3. Open the CoDocs extension from the Chrome toolbar.
4. Choose your preferred formatting options.
5. Apply formatting to instantly generate a syntax-highlighted code block.

---

## Technical Architecture

* **Manifest Specification:** Chrome Extensions Manifest V3
* **Frontend UI:** HTML5, CSS3, and modern browser APIs
* **Core Logic:** Vanilla JavaScript with no external framework dependencies
* **Editor Integration:** Native interaction with Google Docs document content

---

## Privacy and Security

* **Local Processing:** All formatting operations occur entirely within the user's browser.
* **No Data Collection:** CoDocs does not collect, store, or transmit code snippets or document content.
* **No External Requests:** Formatting is performed locally without sending data to external servers.
* **Minimal Permissions:** Only requests permissions necessary for Google Docs integration.
